# VDV301tester

This is a program used to test devices compatible with VDV301 1.0 and VDV301 2.2CZ1.0.

Can be compiled with QT5.15.2
Requirements:
    Windows:
        Bonjour
    Linux:
        Avahi
    
The program is now only available in Czech.
